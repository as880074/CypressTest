# CypressTest



## 安裝 Cypress

安裝前需要先確認電腦有安裝npm可以透過以下指令查詢是否有安裝

```shell

npm -v

```



![image-20230303192425354](.\pitcutre\image-20230303192425354.png)



在使用前，需要使用 npm 安裝 Cypress ：

```shell

npm install cypress --save-dev

```

## 使用 Cypress

在撰寫測試時，可以開啟 GUI ，讓我們可以更直覺地看到測試的狀態：

```shell
npx cypress open
```
